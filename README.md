# course-selecting-system
## 简介
  这是一个小学期课程项目（课程为程序设计导论），这个项目是一个学生选课系统的简单实现
## 生成可执行文件
### Linux
  1. mkdir cmake-build-debug
  2. cd cmake-build-debug
  3. cmake ..
  4. make
### Windows
  使用支持cmake的ide（如VS, QtCreator, Eclipse, CLion）
